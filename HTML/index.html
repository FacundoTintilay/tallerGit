GIT

¿Que es git?

Git es un sistema de control de versiones distribuido utilizado para el seguimiento de cambios en archivos y proyectos de software.


Front-End

¿Que es front-end?

Es la interfaz visual y funcional con la que los usuarios interactúan para realizar acciones, visualizar información y utilizar las características de la aplicación.


Back-End

¿Que es back-end?

Se refiere a la parte de una aplicación que está encargada del procesamiento y la gestión de los datos, así como de las operaciones detrás de escena que hacen que la aplicación funcione de manera eficiente y coherente.
